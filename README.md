# Clon de Netflix 🎬

Este es un clon simple de la página principal de Netflix, hecho con HTML y CSS.  
Incluye diseño responsive para adaptarse a computadoras, tabletas y móviles.

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- Media queries
